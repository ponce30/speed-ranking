読み取り列: Player, Team, H-1st (SEC), 1st-2nd Steal (SEC), SS (M/S), Game — 列順は自由、他の列が混ざってもOK(ヘッダ名で必要列だけ自動抽出)。Tableauで Game セルが複数行に分かれていても自動結合(Gameは末尾の列であること推奨)。Game から日付と対戦カードを抽出して表示。Team の値で選手名を球団色化

ランキング対象: H-1st (秒) と SS (m/s) が有効な行のみ(1st-2nd Steal は欠損OK)。除外条件: H-1st < 3.0 (計測エラー) / SS ≥ 11.0 (人間の限界超え)
